> An exploratory data analytics project applying SQL grouping, filtering, and Python processing to analyze historical user trend datasets and map subscription changes over time in Power BI.

---

## Project Overview

Understanding subscription retention and user behavior trends is critical for business growth. This project examines historical user dataset metrics using SQL queries to group, filter, and aggregate trends, followed by visual reporting in Power BI to highlight key changes over time.

---

## Key Analysis Objectives

1. **Trend Identification:** Use SQL grouping and filtering operations to map changes in subscriber retention.
2. **Dataset Preprocessing:** Clean and structure historical user datasets using Python (Pandas).
3. **Visual Insights:** Build interactive Power BI dashboards to track historical churn and user activity patterns.
